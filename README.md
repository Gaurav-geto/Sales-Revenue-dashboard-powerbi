# Sales and Revenue Dashboard

A Power BI dashboard designed to analyze sales performance, revenue trends, and key business metrics.

## Project Overview

This dashboard provides interactive visualizations to help understand:

* Total Revenue
* Sales Performance
* Revenue Trends
* Product Analysis
* Business Insights

## Tools Used

* Power BI
* Microsoft Excel

## Dataset

The dashboard uses a structured sales dataset stored in Excel format.

## Features

* Interactive filtering
* KPI tracking
* Revenue analysis
* Trend visualization
* Business reporting
